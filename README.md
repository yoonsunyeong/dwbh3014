# 사업장 인사/급여 통합 관리 프로그램

Tkinter + SQLite 기반의 오프라인 인사관리 프로그램입니다.

## 기능
- 근로자 등록/수정/삭제
- 급여 계산(기본급, 연장수당, 상여, 수당, 4대보험/세금 간이 공제)
- 임금대장 조회 및 CSV 내보내기
- 임금명세서 조회 및 텍스트 저장
- 연차 사용 등록, 연차 부여일수 수정, 잔여 연차 확인

## 실행
```bash
python hr_payroll_app.py
```

## 데이터 저장
- 실행 폴더에 `hr_payroll.db` 파일이 생성됩니다.

## 참고
- 공제율(국민연금/건강보험/고용보험/소득세)은 샘플 계산용 기본값입니다.
- 실제 운영 시 회사 규정과 최신 법령에 맞춰 조정하세요.
